# SFCpyxel
## 実行方法
1. リポジトリをクローン
```
git clone https://github.com/umiyuri777/SFCpyxel.git
```
2. ディレクトリ移動
```
cd SFCpyxel
```
3. 仮想環境を設定
```
python3 -m venv venv
```
4. 仮想環境をオン
```
# windowsの場合
.\venv\Scripts\activate

# Macの場合
source venv/bin/activate
```
5. パッケージインストール
```
pip install -r requirements.txt
```
6. 実行
```
python GameController.py
```
